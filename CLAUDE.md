# projectHelm

## Commands

This repo uses **pnpm**. Common scripts:

- `pnpm dev` — start the dev server (port 3000)
- `pnpm build` — production build
- `pnpm test` — run the test suite (vitest)
- `pnpm typecheck` — type-check the project (`tsc --noEmit`)
- `pnpm format` — auto-format the codebase with Prettier
- `pnpm format:check` — verify formatting without writing changes

## Required checks after making changes

**Always** run the following before considering a task complete, and fix any issues they surface:

1. `pnpm typecheck` — must pass with no errors.
2. `pnpm format` — auto-format your changes (or `pnpm format:check` to verify).

Do not leave the working tree with type errors or unformatted code.

## Notes

- `src/routeTree.gen.ts` is generated by TanStack Router — never edit it by hand; it is excluded from formatting.
- Path alias `#/*` and `@/*` both resolve to `./src/*`.
